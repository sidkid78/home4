## Architecture & State Flow
1. **API Initialization**: The constructor securely checks for `GEMINI_API_KEY`. If absent, warns and falls back to a mock mode.
2. **Media Fetch & Parse**: Multimodal inputs are resolved by downloading the target image URLs asynchronously and converting them into base64 encoded binary inline data parts.
3. **Structured Schema Output**: Leverages Google GenAI JSON schema-guided response configuration to yield predictable, highly-structured assessment data objects.
4. **HITL Evaluation**: Triggers the `NEEDS_REVIEW` workflow state automatically if the overall score falls below 0.75 or if any crucial accessibility measurement's confidence is flagged as `low`.
